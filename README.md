## 👨‍💻 Execução

```bash
## 1. Clone o projeto
git clone https://github.com/elanonc/design-system-ignite-lab

## 2. Entre no diretório do projeto
cd design-system-ignite-lab

## 3. Instale as dependências
npm install

## 4. Inicie o storybook
npm run storybook

## 5. Inicie o projeto
npm run dev
```
